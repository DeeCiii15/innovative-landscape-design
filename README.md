# Innovative Landscape Design

Modern Next.js website for [Innovative Landscape Design](https://ildsc.com) — a Florence, SC landscaping company offering design, irrigation, lighting, hardscapes, and maintenance.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

- **Home** — Hero, services overview, about teaser, project gallery
- **Services** — Detailed service descriptions (design, maintenance, lighting, irrigation, hardscapes)
- **About** — Mission, vision, and core values
- **Contact** — Address, phone, email, map, and contact form

## Tech stack

- [Next.js 16](https://nextjs.org) (App Router)
- React 19
- TypeScript
- Tailwind CSS 4

## Images

Project photos and branding assets are sourced from the original ildsc.com site and stored in `public/images/`.
